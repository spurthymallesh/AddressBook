# Address Book Management System

##  Overview

The Address Book Management System is a console-based application developed in C that allows users to efficiently manage contact information. It supports creating, searching, editing, deleting, listing, and saving contacts while performing comprehensive input validation.

---

## Features

- Create new contacts
- Search contacts
- Edit existing contacts
- Delete contacts
- Display all contacts
- Save contacts to a file
- Input validation
- Duplicate contact detection

---
